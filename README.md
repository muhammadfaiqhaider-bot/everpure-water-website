# EverPure Water Website

A modern full-stack website for EverPure Water Plant.

## Tech Stack
- **Frontend:** React.js, Tailwind CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB

## Project Structure
- `client/` : Frontend React application
- `server/` : Backend Express API
